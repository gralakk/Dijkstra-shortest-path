# Dijkstra Shortest Path

## Autorzy
Karolina Gralak, Wiktor Chmiel

## Opis projektu
Projekt z zakresu algorytmów i struktur danych polegający na implementacji algorytmu Dijkstry w celu wyznaczenia najtańszej ścieżki w grafie.

## Cel projektu
Celem projektu było zastosowanie algorytmu Dijkstry do rozwiązania problemu znajdowania najkrótszej, czyli najmniej kosztownej ścieżki pomiędzy wybranymi wierzchołkami w grafie.

## Zakres projektu
- analiza problemu najkrótszej ścieżki,
- reprezentacja grafu w postaci macierzy lub innej struktury danych,
- implementacja algorytmu Dijkstry,
- wyznaczanie kosztu ścieżki,
- analiza poprawności działania algorytmu.
